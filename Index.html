¿No sabes qué hacer?

•Configurar el navegador:

Copia la URL de la barra de búsqueda del navegador y pégala en configuración de página de inicio.

•Realizar una búsqueda:

Al abrir StartPage, la barra de búsqueda quedará enfocada, por lo que ya sólo tendrás que escribir.
Para borrar los datos, puedes oprimir la tecla flecha abajo del teclado.

•Cambiar el proveedor de búsqueda:

Pasa el cursor del ratón en el logo del proveedor de búsqueda, se desplegará un menú de lista donde puedes seleccionar otro proveedor como Google, Bing, YouTube y DuckDuckGo.

•Personalizar:

Abre el menú desde la parte superior izquierda de la pantalla, o también puedes oprimir la tecla flecha arriba del teclado.
Después, selecciona Personalización y cambia el fondo de pantalla, colores y modo de luz.
El fondo de pantalla automático configura una imagen diaria de StartPage, puedes utilizar tu foto favorita como fondo de StartPage, para ello, necesitarás la URL de la imagen en el dispositivo o en la web.

•Cambiar información personal:

Abre el menú desde la parte superior izquierda de la pantalla, o también puedes oprimir la tecla flecha arriba del teclado.
Después, selecciona Configuración y cambia el nombre de usuario y correo electrónico. También puedes cambiar el nombre de usuario desde la parte superior del menú.

•Administrar historial de búsqueda:

Para ver el historial, abre el menú y selecciona Historial.
Para borrar el historial, ve a Configuración y selecciona Borrar historial.

•Mejorar accesibilidad:

Abre la configuración de StartPage, y en Accesibilidad cambia el tamaño de los elementos de la pantalla.

•Compartir StartPage:

Abre el menú y selecciona Compartir, se generará un código QR que te envía al enlace de StartPage en GitHub.

•Restablecer configuración:

Abre la configuración de StartPage, y selecciona Restablecer para volver a la configuración inicial.
Esta acción borra datos personales, configuración e historial de búsqueda.

•Comandos:

Para ejecutar un comando, escríbelo en la barra de búsqueda y presiona "ENTER".
La lista de comandos disponibles son: info, ayuda, historial, borrar historial, restablecer, personalizar, configurar, edge-surf, pacman, snake, t-rex.

•Características:

Esta edición de StartPage funciona con el formato web HTML5, reduciendo el uso de memoria RAM, pero con el mismo nivel de personalización. Está programada para funcionar con dispositivos de pocos recursos de hardware, acelerando la navegación.
Para obtener todas las funciones de StartPage, descarga la versión completa actual (v6.5).
